# 🚀 Design and Evaluation of a Sales Lead Qualification Classification System for Business Growth Analytics

An AI-powered Machine Learning application that predicts whether a sales lead is **Qualified** or **Not Qualified** using Business Growth Analytics. The system helps organizations prioritize potential customers, improve sales efficiency, and increase conversion rates through predictive analytics.

---

# 📌 Project Overview

This project uses the **Random Forest Classifier** to analyze customer and marketing data and determine the likelihood of lead conversion. The Machine Learning model is developed using **Google Colab**, while the frontend dashboard is built with **HTML5, CSS3, and JavaScript**. The complete project is hosted on **GitHub** and deployed using **Vercel**.
